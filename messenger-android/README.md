What is GNUnet Android?
===============

This project contains an Android Studio project for an Android messenger
application which uses the libgnunetchat library 'https://git.gnunet.org/libgnunetchat.git/'.
The app is an uncomplete port of the GTK messenger app 'https://git.gnunet.org/messenger-gtk.git/'.

This is an ALPHA release.  There are known and significant bugs as
well as many missing features in this release.

GNUnet Android is free software released under the GNU Affero General Public
License (v3 or later). For details see the COPYING file in this directory.

Additional documentation about GNUnet can be found at
https://gnunet.org/ and in the 'doc/' folder.
Online documentation is provided at
'https://docs.gnunet.org'.
